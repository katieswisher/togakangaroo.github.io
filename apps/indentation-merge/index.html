When programming indentation is really useful for helping you keep track of things. Usually, if a part of the program is contained in another part, you will want to indent it. 


<p>
	like this!
<p>
	

function anExample() {
	return "or like this";
}

This is true for html, css, and javascript (although what "inside of" means is different for each one).

Practice proper indentation by making the code on the left match the code on the right. Do this by focusing on using the Tab, Enter, and Backspace keys. Don't forget that you can quickly move through through code with the keyboard by using ctrl+<- and ctrl+->.
